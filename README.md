# 🎓 Flashcard Quiz App
An interactive flashcard-based study application built with HTML, CSS, and JavaScript as part of the **Horizon TechX App Development Internship**.

## 🚀 Features
- 🔄 **Flip Cards** — Click card to toggle between Question and Answer
- ⬅➡ **Navigation** — Next/Previous buttons to browse flashcards
- ➕ **Add Flashcard** — Create new custom flashcards
- ✏️ **Edit Flashcard** — Modify existing flashcards
- 🗑️ **Delete Flashcard** — Remove unwanted flashcards
- ✅ **Score Tracker** — "I Know It" / "Don't Know" buttons
- 📊 **Progress Bar** — Visual progress tracking
- 🌙 **Dark/Light Mode** — Toggle between themes
- 🔊 **Sound Effects** — Audio feedback on interactions
- 💾 **localStorage** — Data saved locally, persists on refresh

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- Web Audio API (Sound Effects)
- localStorage (Data Persistence)

## ▶️ How to Run
1. Clone or download this repository
2. Make sure `BG QUIZ.jpeg` and `flashcard-quiz-app-v2.html` are in the same folder
3. Open `flashcard-quiz-app-v2.html` in any browser
4. No installation required!
